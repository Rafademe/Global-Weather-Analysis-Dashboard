WeatherPY


This project, WeatherPy, uses Python to explore possible relationships between weather and latitude. It is generally thought that temperatures increase as you approach the equator but can show it with data? Through pulling weather information from hundreds of positions around the world and plotting this with latitude, we are able to see some trends.

I used the openweathermap.org API to get real time weather information.
